# 🚀 Mentora AI — Smart Academic Assistant

Mentora AI is an AI-powered academic assistant designed to help students learn smarter through features like chatbot support, notes summarization, and quiz generation.

---

## ✨ Features

* 🤖 AI Chatbot for academic queries
* 📄 Notes Summarization
* 🧠 Quiz Generation
* 🔐 User Authentication System
* 📊 Progress Tracking (UI)
* 🎨 Modern UI with Tailwind CSS

---

## ⚠️ Important Notice

> 🚧 **Backend services are currently disabled**

This project uses AI APIs (OpenAI / OpenRouter) that require active billing/subscription.

Due to this:

* AI chatbot responses are **not working**
* Summarization and quiz generation are **disabled**
* Backend API routes are currently inactive

✅ However:

* Frontend UI is fully functional
* Design, layout, and user experience can be explored
* Project structure and integration logic are implemented

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/mentora-ai.git
cd mentora-ai
```

### 2. Install dependencies

```bash
cd client
npm install
npm run dev
```

### 3. Backend (optional / currently inactive)

```bash
cd server
npm install
npm start
```

---

## 🔐 Environment Variables (Backend)

To enable AI features, add:

```
OPENAI_API_KEY=your_api_key
```

⚠️ Requires active subscription/billing.
